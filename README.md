# DevTools

Пройдя этот блок ты освоишь ключевые части DevTools Chrome

- Device Mode
- Elements panel
- Console panel
- Sources panel
- Network panel
- Performance panel
- Memory panel
- Application panel

## Необходимые знания

Базовые знания HTML, CSS и JavaScript

## Самостоятельная подготовка

Установить Google Chrome https://www.google.com/chrome/

## Очная встреча

~ 4 часа
